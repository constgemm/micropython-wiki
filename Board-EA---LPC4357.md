###Features###
* NXP  LPC4357, dual core (M4F, M0), both cores @ 204 MHz
* 1024KB flash ROM, 136KB RAM 
* 164 GPIO
* 2 10bit ADC
* USB, SPI, I2C, UART, 10/100T Ethernet MAC
* LCD controller with dedicated DMA
* 3 phase PWM
